# Network Scanning and Service Enumeration using Nmap
This project demonstrates use Nmap (Network Mapper) for network discovery, host identification, port scanning. Nmap can be used to gather information about systems and services running on a network. Perform host discovery on a network. Identify open ports and running services. Analyze scan results for security assessment purposes.

This project provided hands-on experience with Nmap and demonstrated how network scanning can be used to identify hosts, services, and potential attack surfaces within a network.
