# Nmap-Network-Scanning
Network scanning and host discovery project using Nmap.

Project Overview
This project demonstrates the use of Nmap (Network Mapper) for network discovery, host identification, and port scanning. The objective was to understand how Nmap can be used to gather information about systems and services running on a network.

Objectives

Perform host discovery on a network.
Identify open ports and running services.
Understand basic Nmap scanning techniques.
Analyze scan results for security assessment purposes.

Tools Used
Nmap
Kali Linux
Metasploitable

Activities Performed
1. Host Discovery
Identified active hosts on the network.
Verified network connectivity.
2. Port Scanning
Scanned target systems for open ports.
Identified common services running on discovered ports.
3. Service Enumeration
Gathered information about services running on open ports.
Analyzed scan results to understand the target's exposed services.

Common Commands Used
nmap
nmap -sV
nmap -sn
nmap -p 1-1000
Findings

Successfully discovered active hosts.
Identified open ports and associated services.
Gained practical experience with network reconnaissance and enumeration.

Conclusion
This project provided hands-on experience with Nmap and demonstrated how network scanning can be used to identify hosts, services, and potential attack surfaces within a network.
